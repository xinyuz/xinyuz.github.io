# 通过VuePress来部署GitHub Pages 1/3

# GitHub Pages和VuePress各是什么

GitHub Pages是一个GitHub推出的静态网站托管服务，将HTML，CSS和Javsscript文件从GitHub的repository上读取，然后构建，并发布为一个网站。VuePress是Vue驱动的静态网站生成器，以Markdown为中心的项目结构，以最少的配置帮助你专注于写作。

GitHub Pages解决的是部署的问题，如果不用它，可以自己搭个WordPress的环境，那就需要租用服务器，安装php，MySQL，MySQL Admin等组件。

# 如何部署

  1. 建立一个github repositories，设置为Pages
    1. 新建一个repositories，设为public
    2. 进入Settings，选择Code and automation下的Pages
