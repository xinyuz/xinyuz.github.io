# Git常用命令

# 基础命令

# 1. 在CentOS上安装git

yum install git

# 2. 存储登录信息

如果希望系统记住git用户名和密码，不希望每次都输入验证权限。可以执行这个命令。

git config --global credential.helper store

# 3. 使用代理访问Git Host

git config --global https.proxy http://127.0.0.1:1080

git config --global https.proxy https://127.0.0.1:1080

git config --global --unset http.proxy

git config --global --unset https.proxy

# 4. Git删除远程已不存在的branch

使用过一段时间后，有时在远程已经删除的branch，在本地还可以通过branch -a查到，比如：

% git branch -a
  dev
  main
* story/dajiang-1
  remotes/origin/3
  remotes/origin/HEAD -> origin/main
  remotes/origin/dev
  remotes/origin/issue1
  remotes/origin/main
  remotes/origin/story/dajiang-1

其中的issue1其实就已经删除了。那么如何从本地也清理掉这些已经不存在远程的分支呢？

% git fetch -p
来自 https://hello
 - [已删除]          （无）  -> origin/issue1

在查看git branch -a，就已经没有了

% git branch -a
  dev
  main
* story/dajiang-1
  remotes/origin/3
  remotes/origin/HEAD -> origin/main
  remotes/origin/dev
  remotes/origin/main
  remotes/origin/story/dajiang-1
